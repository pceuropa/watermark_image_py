# Watermark on image

Simple python code help watermark your images with Transparency and positioning of watermark.
Code based on Python3 and [Pillow](https://pillow.readthedocs.io/en/stable/)


INSTALATION
----------
```bash
pip3 install -r requirements.txt
```

USAGE
----------
```bash
python3 main.py
```
